# 🎓 Online Course Platform

A simple online learning platform where mentors can create courses, users can take exams and receive certificates, and administrators manage the system.

---

## 🚀 Project Overview

This project is a course management platform with three types of users:

- **Admin**
- **Mentor**
- **User**

Mentors can submit courses, admins review and approve them, and users can explore courses, take exams, and earn certificates.

---

## 👤 User Roles

### 🔹 Admin
- Approve or reject courses submitted by mentors
- Manage users (edit / delete / assign roles)
- Create and manage exams
- Write, edit, and delete blog posts
- Manage complaints from users
- Respond to user feedback

### 🔹 Mentor
- Create courses
- Submit courses for admin approval
- Manage personal profile
- Receive ratings and comments from users

### 🔹 User
- Browse available courses
- Take exams (paid or free)
- Receive certificates after completing exams
- Rate mentors with star ratings
- Leave comments on mentor profiles
- View earned certificates in the profile

---

## ⚙️ Core Features

- Role-based authentication system
- Course submission and approval workflow
- Exam system
- Certificate generation
- Mentor rating and comment system
- Blog system managed by admin
- Complaint management system
- CRUD operations for users, courses, and profiles
